# LearnX Dashboard

A futuristic student learning dashboard built with:

- Next.js App Router
- Supabase
- Tailwind CSS
- Framer Motion
- TypeScript

## Features

- Server-side Supabase data fetching
- Bento Grid Layout
- Animated Course Cards
- Responsive Design
- Skeleton Loading States
- Error Boundaries
- Framer Motion Interactions

## Setup

npm install

npm run dev

## Environment Variables

Create .env.local

NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
